Projeto Access

Um projeto que visa aumentar a acessibilidade para deficientes auditivos em plataformas digitais.

Sobre

Apaixonados por tecnologia lutando por um mundo melhor

A ideia do projeto é criar tecnologias de acessibilidade como algoritmos de transcrição de vídeo para texto, legendas de conteúdo ao vivo e gravado, além de tradução em LIBRAS (Língua Brasileira de Sinais). Além disso, pretendemos criar um portal de informações sobre acessibilidade e criar programas interativos para o ensino de LIBRAS. Este projeto surgiu a partir de um relato de uma aluna surda que não estava tendo acesso às aulas online devido à falta de acessibilidade da plataforma. O Access se propõe a contribuir para o objetivo de desenvolvimento 4 da ONU: Educação de Qualidade. No cenário de 2020, o ensino teve que se adaptar às atividades remotas. Por meio da nossa plataforma, acreditamos que tornaremos o espaço digital, principalmente as aulas e reuniões online, mais inclusivo e que poderemos contribuir para a democratização do acesso à informação. O Projeto Access conquistou o Prêmio Desafio Criativos da Escola, no qual recebemos uma verba para investir no Projeto e maior visibilidade.

Missão

Criar uma sociedade mais acessível para a Comunidade Surda

Disponibilizar ferramentas para tornar o meio digital mais acessível para a comunidade surda.

Visão

Ser a maior plataforma de acessibilidade

Queremos que até 2030 a maioria dos vídeos na web sejam acessíveis a todos os públicos.


Valores

Acessibilidade

- Humanidade

- Inclusão

- Respeito

- Paixão por tecnologia
